## What Changed

This commit introduces Phase 2 foundational components to SplatRAG v1, bumping the status to v1.1. It adds real-time consolidation during generation (micro-dreams), CLI controls, and an evaluation sweep harness.

### micro-dreams (src/dream.rs, src/main.rs)
- Added `micro_dream` function to `dream.rs`. It provides real-time consolidation during generation.
- It operates by taking short forward and backward physics bursts. It forward projects the current position 2 steps into the future, backward anchors the projection toward the goal, and computes a correction delta.
- The correction is scaled by a `blend_factor` of `0.10` and added to the current position to provide a gentle nudge.
- In `main.rs`, this is triggered on the 4096-dimensional slice every 8 steps after step 5.

### Persistent Memory & CLI (src/main.rs)
- Added CLI arguments `--prompt`, `--model`, and `--clear-memory`.
- Automatically loads persistent splat memory from `data/splat_memory.safetensors` unless `--clear-memory` is provided, enabling memories to survive across separate runs.

### Evaluation Harness (sweep.sh)
- Created `sweep.sh` which iterates through 5 category prompts (Physics, Technical, Creative, Reasoning, Abstract) and parses the token count and output.
