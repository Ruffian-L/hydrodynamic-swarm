- **`memory.rs`**: The `has_nearby` function, which prevents splat clumping, was previously scanning the entire splat array (O(N)). It now only samples the last 50 splats, significantly reducing CPU overhead as the memory grows.
- **`main.rs`**: The top-5 attractor token search previously sorted the entire 128K vocabulary array. It now uses a partial sort (`select_nth_unstable_by`) to isolate the top 5 elements in O(N) time without fully sorting the array.

## Physics Impact

- **Steering (tui.rs)**: The chat mode introduces *online* splat creation during generation. The sigma parameter is dynamically scaled based on the severity of the steering delta (e.g., `delta_norm > 30.0` -> `sigma = 70.0`), creating massive gravitational scars when the model requires a massive correction.
- **Memory (memory.rs)**: Bounding the `has_nearby` check to the last 50 splats assumes temporal locality—that recent memories are the most likely to collide during a single continuous generation. This trades perfect field uniformity for the raw TPS speed needed for real-time chat.
- **Dreaming (tui.rs)**: Adaptive micro-dreams are now active in the chat loop. The engine analyzes the raw softmax probabilities for entropy. High entropy (`> 4.0`) triggers a deeper 4-step forward projection (recursive collision) to synthesize context before continuing.

## Key Findings

1. **The Cost of Sorting**
   - Evidence: Replacing the full sort with `select_nth_unstable_by` for the 128K element probability array.
   - Significance: As the context window and vocabulary size increase, simple operations like finding the top-K attractors become catastrophic bottlenecks. O(1) or O(N) partial operations are essential for maintaining the real-time illusion of the swarm.

2. **Temporal Locality in Memory Space**
   - Evidence: `has_nearby` only checking the last 50 splats.
   - Significance: During a single prompt trajectory, the model's path through the latent space is continuous. It only risks clumping with splats it *just* created, allowing us to drop the O(N) distance check against historical memories.
